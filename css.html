Тэги, селекторы, атрибуты:

  • style. В нём:
    1. элементный селектор (например, "p")
    2. или селектор класса (например, ".classname")
    3. или селектор по id (например, "#id")
    4. или универсальный селектор "*"
    5. или группировка селекторов (например, "h1, h2, p")
    6. или другие селекторы: >(перейти вглубь по селектору),
      a+b(след.элемент b после a),
      a~b(элемент b перед a),
      элемент:first-child (первый элемент внутри другого),
      input[type="text"] {} (стиль для всех элементов input, у которых атрибут type="text")
      a:hover{} (стиль для ссылки при наведении на нее курсора мыши.)
      p::first-line {} (стиль для первой строки в каждом элементе p)
    7. далее - в фигурных скобках свойства ("{атрибут:значение}")
  • color: blue - установить синий цвет
  • background: linear-gradient(to right, blue, red) - градиент
  • text-align: center - установить выравнивание по-центру
  • font-size: 26px - установить развер шрифта 26пикс (pt - в пунктах; 1.2em - 1.2 от размера шрифта родит. элемента; 1.2rem - от размера шрифта корневого элемента.)
  • line-height: 1.5 - установить высоту строки
  • font-weight: насыщенность шрифта
  • font-style: стиль шрифта
  • font-family: "шрифт1", "шрифт2",...: шрифты
  • link rel="stylesheet" href="styles.css" - подключить стиль (stylesheet - таблица стилей)
  • b - выделить текст жирным (строчный элемент). Внутри элемента можно установить стиль
  • u - подчеркнутый текст (строчный элемент)
  • div - блочный контейнер для группировки содержимого и структурирования страницы. (атрибуты - class, id, style)
  • figure - контейнер для аннотации различных иллюстраций. В него вкладывается, например, figcaption (заголовок) и img
  • span - основной контейнер для группировки строчного контента
  • text-decoration - свойство стиля, позволяющее, например, подчеркнуть текст
  • тэг "!important" - задать наивысший приоритет стилю.
  • Позиционирование.
    1. position: static - статическое (по умолчанию - порядок друг за другом)
    2. position: relative - относительное (относительно предыдущего элемента)
    3. position: absolute - абсолютное (позиционирование ближайшего предка с позиционированием, отличным от статического.)
    4. position: fixed - фиксированное (относительно окна браузера, независимо от прокрутки)
    5. transform: translate(x,y) — перемещает элемент на заданные координаты;
    6. transform: scale(1.05); - скейл масштаба элемента
    7. left: 12px — положение от левого края
    8. top: 12px — положение от верхнего края
    9. z-index: 10 - позиционирование по слоям
    10. Работа с процентами (width: 30% - от ширины экрана, left: 30% - отступ от ширины экрана)
    11. width: calc(50% - 20px); - расчет ширины элемента11
  • Отображение
    1. display: block - занять всю ширину и начать новую строку
    2. display: inline - занять минимум места и не начинать новую строку
    3. display: inline-block - блок с возможностью задания ширины и высоты
    4. display: none - скрыть элемент
    5. float: left - расположить элемент слева и задать обтекание его справа
    6. clear: left - удалить свойство "left"
  • @media screen and (max-width: 400px) and (min-width: 200px){элемент{}...} - медиа запрос. применить свойства к экрану, у которого ширина не более 400 пикс.
  • Сетка
    1. .grid-container {
      display: grid;
      grid-template-columns: 1fr 1fr (или repeat(2, 1fr)); - столбцы
      grid-template-rows: 100px 100px 100px 300px (или 1fr 1fr...); - строки
      grid-template 10px 10px / 10px 10px 10px; - 2 строки, 3 столбца (вместо первых двух команд)
      gap: 10px (или 10px 20px); (расстояние между ячейками)
      }
    2. grid-template-columns: 1fr 1fr 1fr — вся доступная ширина разделится на 3 части, и каждая колонка займёт одну из этих частей
    3. auto - минимальный размер
    4. minmax(200px, 300px) - задать ширину ячейки 200-300px
    5. Элементы грида
      grid-column: span 2; -соединить 2 столбца
    6. grid-column: span 2 - соединить 2 ячейки
  • Flexbox - гибкий контейнер. Строится на div(контейнер), в который вложены div(элементы). В стиле контейнера устанавливается display: flex; (Информация по Flex.)
    1. display: flex; - поведение блочного элемента.
      display: inline-flex; - поведение строчного элемента.
    2. Приоритет ширины контейнера: min-width>flex-basis>width>content
    3. gap: 10px; - промежуток между ячейками (устанавливается в контейнере)
    4. justify-content: space-between - Применяется к родительскому элементу. Флексы равномерно распределяются по всей строке.
      justify-content: center - Применяется к родительскому элементу. Флексы выравниваются по центру строки.
      ; justify-content: flex-start - Применяется к родительскому элементу. Флексы прижаты к началу строки.
      justify-content: space-around - флекс-элементы равномерно распределяются по всей строке;
    5. align-items: flex-start; (свойство контейнера)- Флексы выравниваются в начале поперечной оси контейнера.
      align-items: stretch; (свойство контейнера)- Флексы растягиваются таким образом, чтобы занять всё доступное пространство контейнера.
    6. align-self: flex-start; - cвойство align-self выравнивает флекс-элементы текущей строки, переписывая значение align-items.
    7. flex-direction: column - Элементы располагаются сверху вниз (имеется column,...)
    8. flex-wrap: nowrap; - элементы только в одну строку
      flex-wrap: wrap - элементы могут переноситься на следующие строки;
    9. align-content: flex-start; (свойство контейнера) - ряды группируются в верхней части контейнера. (! работает только когда у контейнера есть свойство flex-wrap: wrap;)
    10. flex-flow: column wrap представляет собой сокращённую запись свойств flex-direction и flex-wrap
    11. flex-grow: 0/1 - запретить/разрешить растяжение элемента
    12. flex-shrink: 0; - запретить уменьшение элемента (задает коэффициент сжатия flex с заданным числом)
    13. flex-basis: 100px - Начальный размер элемента. Сделать контейнеры шириной 100 px. При уменьшении размера страницы гарантирует оптимальное изменение размера контейнеров.
    14. flex: 1 1 0; - задать одновременно flex-grow, flex-shrink, flex-basis (элемент может расти, сжиматься и имеет начальный размер в зависимости от контента)
    15. order: 0 - порядок отображения (элемент с этим свойством выводится самый первый)
